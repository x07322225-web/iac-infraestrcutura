# Infraestructura como Código

Este repositorio contiene scripts de infraestructura como código utilizando Terraform.

## Recursos creados

- Red (VPC y Subnet)
- Recurso de cómputo (Instancia)
- Almacenamiento (Bucket)

## Requisitos

Tener instalado:

- Terraform
- Git
- Cuenta en AWS

## Ejecución

1. Clonar repositorio

git clone https://github.com/x07322225-web/iac-infraestructura.git

2. Entrar al proyecto

cd iac-infraestructura

3. Inicializar terraform

terraform init

4. Ver el plan

terraform plan

5. Crear la infraestructura

terraform apply
